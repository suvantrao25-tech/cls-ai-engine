# CLS AI Engine - Frontend

This folder contains the **user interface (dashboard)** for CLS AI Engine.

## Purpose

The frontend is the control panel where users can:

- Create new AI content projects
- Enter tool or website URL
- Generate SEO blogs automatically
- Manage affiliate links
- View generated content drafts
- Publish content to WordPress
- Schedule social media posts

---

## Core Features (Planned)

### 1. Dashboard
- Overview of generated content
- Traffic & performance stats
- Recent projects

### 2. New Project Panel
- Input: Tool name or URL
- Category selection (AI Tools, SaaS, etc.)
- Generate button

### 3. Content Generator UI
- SEO article preview
- Edit before publish
- Meta title & description editor

### 4. Publishing Panel
- WordPress connect
- One-click publish
- Draft management

### 5. Social Media Panel
- Auto-generated posts
- Schedule posts to:
  - Facebook
  - Instagram
  - LinkedIn
  - X (Twitter)
  - Pinterest

---

## Tech Stack (Planned)

- Next.js (React Framework)
- Tailwind CSS (UI Styling)
- Axios (API calls)
- Zustand / Redux (State management)

---

## UI Vision

Simple, fast, and creator-focused dashboard:

> "One tool → Full content system"
