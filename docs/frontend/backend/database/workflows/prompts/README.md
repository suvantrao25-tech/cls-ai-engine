# CLS AI Engine - Prompts Library

This folder contains the **AI prompt system** for CLS AI Engine.

Prompts define how AI agents behave, think, and generate outputs across the entire system.

---

## Purpose

The prompt system ensures:

- Consistent AI output quality
- Structured SEO content generation
- Accurate affiliate integration
- Standardized writing style
- Multi-agent coordination

---

## Core Idea

Each AI Agent works using a **specialized prompt template**.

> Better Prompt = Better Output = Better Business Results

---

## Main Prompt Categories

---

### 1. Research Agent Prompt

**Purpose:** Gather structured information about a tool or website.

```text id="pm1"
You are a research assistant.

Analyze the given tool/website and extract:

- Overview
- Key features
- Pricing details
- Pros and Cons
- Target audience
- Competitors
- Use cases
- FAQs

Keep the output structured and easy to use for SEO content creation.
